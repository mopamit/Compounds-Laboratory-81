# מעבדת התרכובות

אפליקציית React לימודית לכיתה ח׳ בנושא תרכובות, פירוק, ספירת אטומים וחוק שימור המסה.

## הרצה מקומית

```bash
npm install
npm run dev
```

## פרסום ב-GitHub Pages

1. צרו מאגר חדש ב-GitHub.
2. העלו אליו את כל הקבצים והתיקיות בפרויקט הזה.
3. ודאו ששם הענף הראשי הוא `main`.
4. במאגר פתחו **Settings → Pages**.
5. תחת **Build and deployment → Source** בחרו **GitHub Actions**.
6. לאחר שהפעולה תסתיים, קישור האתר יופיע במסך Pages.

הפרויקט כולל Workflow אוטומטי לפרסום בכל העלאה לענף `main`.
